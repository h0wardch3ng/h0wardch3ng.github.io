October 1, 2017

海盜灣（英語：The Pirate Bay，縮寫：TPB）是一個專門儲存、分類及搜尋 Bittorrent 種子檔案的網站，並且號稱是「世界最大的BitTorrent tracker（BT 種子伺服器）」，提供的BT種子除了有自由版權的收集外，也有不少被著作人聲稱擁有版權的音訊、視訊、應用軟件與電子遊戲等，為網絡分享與下載的重要網站之一。

各國對海盜灣的態度

出於政治和版權保護的問題，中國大陸曾經將海盜灣列為網絡封鎖物件，但可通過 https 加密方式存取。

2009年4月17日，瑞典斯德哥爾摩的法院以侵犯版權罪判處海盜灣四名創始人各一年的監禁，並處以約3000萬瑞典克朗的罰金。

2013年7月，中國大陸已可以正常進入該網站，並自由瀏覽。此次解封可能是因為海盜灣更換了網站域名。

2014年1月 , 中國大陸部分地區再次不可以正常進入該網站，之後又了恢復存取。美國政府向瑞典政府威脅稱，如果瑞典無法解決「海盜灣」問題，將把其列入 WTO 黑名單。

2014年12月10日，海盜灣網站的伺服器遭到了瑞典警方的查封，並關閉網站。2015年2月1日 重新上線。

2015年5月，瑞典一家法院作出判決：要求海盜灣交出 piratebay.se 域名。

2017年2月初，瑞典一家法院做出裁定，命令當地互聯網提供商在接下來的三年時間遮蔽海盜灣。

2017年6月下旬，歐盟法院做出裁定：互聯網提供商有義務根據版權所有者的要求限制用戶存取海盜灣。

2017年9月，荷蘭最高法院裁定：當地互聯網服務供應商 Ziggo 和 Xs4all 在接下來的10天內全面遮蔽海盜灣網站。

思考

關於海盜灣的判罰只是在下載頁面簡單統計了下載的次數來表示版權擁有者所造成的損失，P2P File Sharing 還引出了另外第一個問題：互聯網隱私的政府監控問題,P2P 技術能很好的實現互聯網的去中心化，隱藏用戶與服務器的直接關聯。時至今日，Bitcoin / IPFS / Zeronet / Mastodon / Firechat 等技術，當政府關閉互聯網的鏈接，這些去中心化的應用還能繼續工作。

Firechat 就是一個去中心化的典型例子，詳細見報導

當香港手機網路訊號中斷，FireChat這款「沒有訊號也能聯繫」的APP就此爆紅

FireChat, darling of 0ccupy Centr@l protesters, throws down gauntlet to WhatsApp with off-the-grid texting

不過在大陸地區，P2P 技術已經被玩壞了，迅雷等下載軟件只下載，不上傳的行為嚴重影響了 P2P 網絡的傳輸，就好比一個大水桶，只會從桶裡舀水，而不會往桶裡倒水，這樣總有一天變成一個空桶。

推薦大家使用規範的 P2P 軟件進行下載，例如 Qbittorrent / Transmission / BitComet / BiglyBT / Aria2 / Xmule / Koinonein，下載完成後記得 seed （做種）自己已完成的文件，遵守良好的 P2P 網絡規則。
