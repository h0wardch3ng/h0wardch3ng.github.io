Nexus 5X 更新基於 Android 9.0 Pie 的第三方系統

February 14, 2019

Nexus 5X 的刷機就需要幾個步驟就可以了

  1. 配置 ADB 環境變量（SDK）
  2. 按照最新工廠鏡像做為底包（如果是最新官方系統可忽略此步驟，如果是舊版本系統 7.x/6.x 會驗證 vendor images 會導致無法正常開機）
  3. 準備刷機 ROM 包+ GAPPS + SU（Root 可選）
  4. 安裝手機驅動
  5. 解鎖 BootLoader
  6. 通過 Recovery 卡刷 Rom

配置 ADB 環境變量（SDK）#

現在 Google 也放出了單獨的 ADB 文件了，還是比較方便的

https://developer.android.com/studio/releases/platform-tools

下載完成后記得配 PATH 即可。

安裝工廠鏡像（可選）#

https://developers.google.com/android/images

下載 “bullhead” for Nexus 5X 的 8.1.0 (OPM4.171019.016.A1, May 2018) 壓縮包

下載完成解壓后雙擊脚本開始自動按照工廠鏡像

之後開機進入桌面準備刷機包

AospExtend 系統刷機包 + GAPPS#

AospExtend 基於 Android 9.0 Pie#

https://downloads.aospextended.com/bullhead

Open GApps#

https://opengapps.org/

下載 ARM64 + 9.0 + PICO 或 STOCK

按照手機驅動和解鎖 Bootloader#

下載 XDA 大神的 TOOL ALL IN ONE

點擊界面的 Install Drivers，一路 Next，等待驅動安裝完成

之後點擊 Bootloader 裡面的 Unlock，等待手機重啟

通過 Recovery 卡刷 Rom#

然後手機再次進入 Bootloader 界面，此時使用工具 Recovery 裡面的 Recovery Flasher 刷入 TWRP Recovery 即可

之後進入 TWRP Recovery，記得雙清後刷入刷機包和其他組件
